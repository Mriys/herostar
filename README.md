# HeroStar Frontend

Sarı-beyaz-siyah temalı fitness ekipmanları e-ticaret sitesi (frontend).

## Çalıştırma
1. Bağımlılıkları kurun:
```
npm install
```
2. Geliştirme sunucusu:
```
npm run dev
```

## Yapı
- React + Vite
- TailwindCSS
- React Router
- Basit sepet durumu (Context) – sadece frontend

## Renkler
- Marka: #FFD000 (sarı)
- Arka plan: #0a0a0a (siyah)
- Metin: #ffffff (beyaz)
